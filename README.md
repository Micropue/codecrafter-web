## 基于vue开发的论坛系统

### 项目部署：

```bash
npm i
npm run dev
```

### 项目`src`目录

1.`/api`为存储类型定义、全局配置、封装接口

2.`/assets`为存储项目的静态资源

3.`/components`为存储小型组件「包含相关功能」

4.`/error`错误页面的组价存储位置

5.`/store` 全局状态管理，使用`vuex`（因为`pinia`不好使）

6.`/view`所有页面组件

> [!TIP]
>
> 大部分代码都有注释，一看就懂
